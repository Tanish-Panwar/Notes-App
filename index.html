About Hello Notes

If you are using hello notes for taking notes, you are saving million of trees and money. Hello Notes provides you a note taking facility for absolutely free of cost, you can store 5000+ number of notes using this web app, It is very simple to use app, Add a title add a description and just click the add note button.

USAGE:

1. Add a title for your note, title looks great.

2. Add a Description for your note, so note would be expressive.

3. Click on the "Add note" button done

4. You can search your notes with the given search box just type in your note's letter

5. You can also use + to search your notes

6. You can use styling for your notes, <strong> Bold text </strong> for bold.

7. <em> Italic text </em> for italic.

8. <u> Underline text </u> for underline.

Privacy Policy

We don't store any kind of data, if you are using our web application which is free to use, when you add a note, it is only visible to you because it is stored in your web browser, which means no one else can read and view your notes, you can store any number of notes using Hello Notes, Once again: We don't store any kind of data, we only use 2 cookies which is browser's local storage and session storage for storing notes and for search engine optimization,
